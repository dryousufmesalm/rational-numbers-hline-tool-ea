## Requirements

- MetaTrader 5

## Installation

Copy `numerical tools.mq5` into `MQL5/Experts/` or a subfolder.

## Compilation

Open in MetaEditor and compile (F7).

## Usage

Attach to a chart. Use for rational-number and horizontal-line utilities as designed in the EA.
